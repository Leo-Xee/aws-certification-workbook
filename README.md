# AWS Certification Workbook

AWS Certification 시험 준비를 위한 문제 풀이 애플리케이션입니다.

## 기술 스택

- React 19
- TypeScript
- Tailwind CSS v4

## 개발 환경

### 설치

```sh
pnpm install
```

### 실행

```sh
pnpm dev
```

## Docker

### 이미지 빌드

```bash
# Docker 이미지 빌드
docker build -t aws-certification-workbook .
```

### 컨테이너 실행

```bash
# 컨테이너 실행
docker run -p 3000:80 aws-certification-workbook
```
